History

This page will contain the release history for gentoo. The approximate date for each release is listed, together with some small notes on why the release was made, any big changes/fixes made to the code, and so on. I love reading stuff like this, so I might just write some of my own. :)

v0.0.0
1998-05-17		 Development begins
Right out of the blue, I start playing around with GTK+, trying to put together something that resembles a file manager. It's my first GTK+ program ever. It kind of grew, I guess...
v0.9.0
1998-09-11		   Initial release
Autumn semester has started, so we got to get the code out to the people. :)
